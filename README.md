# 强化学习入门工具与实战 Tips

这是一个面向强化学习初学者的工具与环境配置笔记仓库，重点整理了从开发环境搭建到训练实战中常见问题的处理方法。

目前内容主要覆盖：

- PyCharm 基础使用
- Anaconda / Conda 环境配置与管理
- 原生 Python `.venv` 虚拟环境
- PyTorch、CUDA、cuDNN 安装与排查
- Git 与 GitHub 常见操作
- Claude Code、Codex、Gemini 等 CLI 配置
- 强化学习训练中的随机种子、噪声等实战经验

## 文档入口

完整内容请查看：

- [RL_tips.md](./RL_tips.md)

## 文档主要内容

`RL_tips.md` 当前主要包含以下几个部分：

1. PyCharm 的基础使用与解释器配置
2. Anaconda/Conda 的安装、换源、环境管理与常见问题
3. 原生 Python `.venv` 虚拟环境的创建、激活与清理
4. PyTorch、CUDA、cuDNN、cudatoolkit 的基础概念与安装说明
5. Git、子模块、远程仓库、GitHub Desktop 等常用操作
6. Claude Code、Codex、Gemini CLI 的安装与配置
7. 强化学习训练中的实战笔记，如随机种子与噪声

## 适合谁看

这个仓库更适合以下读者：

- 刚开始接触强化学习，需要先把开发环境搭起来的人
- 在 Windows 环境下使用 PyCharm、Conda、PyTorch 时经常遇到问题的人
- 想整理一份自己的 RL 开发工具链备忘录的人

## 使用建议

- 如果你想快速查问题，建议直接在 [RL_tips.md](./RL_tips.md) 中搜索关键词。
- 如果你正在配置环境，建议优先阅读 Conda、PyTorch、Git 这几部分。
- 仓库内容以经验总结和操作记录为主，后续会继续更新和补充。

## 仓库说明

- 文档以 Windows 环境下的实际使用经验为主
- 部分示例路径、版本号、截图来自作者本机环境
- 使用前建议结合你自己的 Python、CUDA、PyTorch 版本进行核对
